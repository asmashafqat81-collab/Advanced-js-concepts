Spread Operator:
The spread operator spraed or expands elements from arrays or objects, provides a concise way to copy, merge arrays or objects.


Destructuring:
Destructuring allows to  extract values from arrays or objects into variables.Destructuring provides a shorter way to extract data.It works with arrays  and objects both.


Prototype chaining
The prototype chain is inheritance mechanism in JS.Every object can inherit properties and methods from another object through a chain of prototypes.It uses method setpeototypeof .JavaScript works by checking the object itself.If not found, it checks the object prototype.If still not found, it continues up the chain.The search ends when the chain reaches null.


Iterator
An iterator is an object that provides a standard way to access elements of a collection one at a time.An iterator must implement a method called next().Each call to next() returns an object containing: value  and done that indicates whether iteration is complete or not.


Generator
A generator is a special type of function that can pause execution and later resume from the same point.Generators simplify the creation of iterators.
